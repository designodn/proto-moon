# Картинки онбординга «Фотомарафон»

Слоты для компонента `components/onboarding-marathon.{css,js}`.
Положи сюда файлы с этими именами (PNG с прозрачным фоном):

| Файл | Где используется |
|------|------------------|
| _(слайд «Участвуйте»)_ | Веер фото берётся из `data/marathon.json` (фото первых 3 работ) — отдельные png не нужны |
| `icon-sport.png`   | 3D-иконка к чипу «спорт» (гиря) |
| `icon-cooking.png` | 3D-иконка к чипу «кулинария» (пицца) |
| `icon-garden.png`  | 3D-иконка к чипу «сад» (росток) |
| `stickers.png`     | Стикеры в блоке «Голосуйте за фото» (смайл + палец + сердце, одной картинкой, ~340×340) |
| `invite.png`       | Одна широкая картинка в блоке «Приглашайте всех» (image 510, ~432×241, во всю ширину с бликидом) |

Имена файлов жёстко зашиты в `onboarding-marathon.js` (функция `img()` и разметка) —
если поменяешь нейминг, поправь и там.

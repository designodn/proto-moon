# Ассеты виджета «Бонсай»

Иллюстрации для виджета бонсая в профиле New Vision.

Виджет будущем станет компонентом с вариантами и динамикой:
посетители профиля «поливают цветочек» / «чинят машинку», и по мере
взаимодействий иллюстрация в виджете меняется (стадии роста).

## Варианты
- `flower/`  — цветочек (поливают)
- `car/`     — машинка (чинят)
- `green/`   — парник / теплица

## Стадии (по уровням)
Внутри каждого варианта — кадры по уровням, например:
`flower-1.png`, `flower-2.png`, … (росток → деревце → цветущий).

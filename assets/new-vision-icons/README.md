# assets/new-vision-icons

Пак иконок **New Vision** — отдельный от общего `assets/icons` (другой визуальный
язык, другой размер). Сюда кладём только то, что у NV расходится с Q3.

## Таб-бар (первый потребитель)

Иконки подключаются через модификатор `.tabbar.__vision`
(см. `new-vision/components/nv-tabbar.css`). Каждый слот — одна SVG, рисуется
маской (`mask`), поэтому цвет берётся из `currentColor` (активная — оранжевая),
заливка внутри самого SVG не важна.

По макету Figma (node 2855:140933) у NV **5 слотов**, без «Обсуждения»:

| слот      | файл          |
|-----------|---------------|
| `feed`    | `feed.svg`    |
| `book`    | `book.svg`    |
| `message` | `message.svg` |
| `clip`    | `clip.svg`    |
| `menu`    | `menu.svg`    |

- Вьюпорт SVG — **32×32** (у NV иконки крупнее, чем 24 в Q3).
- Если у активного состояния отдельный «залитый» глиф — добавь `feed-filled.svg`
  и т.п., скажи мне, пропишу для `.__state-on`. Пока активная иконка — тот же
  глиф, перекрашенный в оранжевый.

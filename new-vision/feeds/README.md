# new-vision/feeds/

Фид-карточки New Vision, которые **расходятся** с q3.

Принцип: общие примитивы (токены, кнопки, аватары, actions-bar) берём из общей
DS; здесь лежит только то, что в NV выглядит/ведёт себя иначе.

## Конвенция

- Один компонент = один CSS-файл, имя с префиксом `nv-`:
  `nv-feed-base.css`, `nv-feed-text.css`, `nv-feed-discussion.css`, …
- Подключение — через `../new-vision.css` (добавляй `@import` в секцию 2).
- Если NV-карточка лишь чуть меняет общую — наследуй класс общего компонента
  и перебивай только нужные свойства, не копируй файл целиком.

## Подключение нового фида

1. Создай `new-vision/feeds/nv-<name>.css`.
2. Добавь `@import url('./feeds/nv-<name>.css');` в `new-vision/new-vision.css`.
3. Используй разметку на странице `new-vision/lenta.html`.
